Engine Simulator – BMEP-Based Toy Dyno

Build an engine concept, and this model estimates torque, power, BMEP, airflow, and fuel use across the RPM band. It’s driven by BMEP → torque → horsepower, not magic constants. Educational use only – not a live tuning tool.
